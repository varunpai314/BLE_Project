## Product Naming Philosophy
The system is **not Wenlock-specific**. It is a generic BLE indoor patient tracking platform
that can be deployed in any hospital. The name should reflect that.

**Criteria:**
- Not tied to any hospital/location name
- Works globally
- Sounds professional for hospital pitch deck AND KDEM/Solution Challenge
- Short, memorable, domain-available
- Conveys: location + healthcare + intelligence

## Product Name Options

| Name | Meaning | Vibe |
|------|---------|------|
| **ZonePulse** | Zone tracking + pulse (heartbeat) | Energetic, medical |
| **TraceFlow** | Tracing patient flow | Clinical, clean |
| **LocusCare** | Locus (Latin: location) + Care | Elegant, professional |
| **WardIQ** | Ward intelligence | Smart, direct |
| **Vigilant** / **Vigil** | Watchfulness | Powerful, meaningful |
| **Proximia** | Proximity + -ia suffix | Scientific, unique |
| **PinPoint Health** | Precision location | Clear, modern |

**✅ CONFIRMED: PROXIMIA**
- From *proximity* — the core technology
- Scientific, premium, globally deployable feel
- Not tied to any hospital or location
- Domain: `proximia.in` ✅ available
- Logo: `docs/assets/proximia_logo_v1.png`
